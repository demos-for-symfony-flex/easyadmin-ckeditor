# easyadmin-ckeditor

## Requirements
### ZipArchive (Zip extension)
* Error thrown while running command "ckeditor:install". Message: "Class 'ZipArchive' not found"
* debian: php7.1-zip, php5.6-zip
* http://www.php.net/manual/en/book.zip.php
